# README
## Code
This folder contains the code required to reproduce this study. The file access is designed to follow this project's folder structure and works best if this Github page us 'pulled' into a new R environment using git (refs).

Each quarto document is designed to act independently as its own study to examine trends in raw material ('SBF_RawMaterial.qmd'), flakes ('SBF_flakes.qmd'), cores ('SBF_Cores.qmd'), scrapers ('SBF_Scrapers'), and comparing aggregation trends ('SBF_AggregationComparison.qmd').
