# Results
This folder contains three sub-folders:

- 'figures':
This folder contains the figures that are exported from the Quarto documents in the '/code' folder. The figures are named in chronological order as they appear in the publication. Not all figures are produced in R.

- 'plates':
This folder contains a sample of artifact plates to highlight the variation in flake, core, and scraper forms and morphology. Images and vectors were processed in Adobe Photoshop and Adobe Illustrator.


- 'tables':
This folder contains tables that appear in the publication. The tabulated data is based on the data in the Quarto document, but the tables were produced outside of R.
